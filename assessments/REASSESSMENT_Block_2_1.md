## ЗАДАНИЕ 1: Предсказать результаты

Что выведет этот код? Напиши ответ + объяснение в одной строке.

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)  # ?
print(a is b)  # ?
print(a is c)  # ?
```

**Твой ответ:**

```
print(a == b)  # ответ: True
объяснение: потому что списки a и b содержат одинаковые значения

print(a is b)  # ответ: False
объяснение: потому что у списков a и b разные места в памяти

print(a is c)  # ответ: True
объяснение: потому что списки a и c ссылаются на один и тот же объект
```

## ЗАДАНИЕ 2: Интернирование

Что выведет? Почему разные результаты?

```python
x = 100
y = 100
print(x is y)  # ?

a = 300
b = 300
print(a is b)  # ?
```

**Твой ответ:**

```
x = 100; y = 100 → x is y = True
объяснение: Питон кэширует малые числа

a = 300; b = 300 → a is b = False
объяснение: а это уже не малые числа, и они уже не кешируются
```

## ЗАДАНИЕ 3: Практика (мини-сценарий)

Прочитай и объясни:

```python
list1 = [10, 20]
list2 = [10, 20]
list3 = list1

print(list1 == list2)      # ?
print(list1 is list2)      # ?
print(list1 is list3)      # ?
print(id(list1) == id(list3))  # ?
```

**Твой ответ:**

```
list1 == list2 → True
объяснение: сравнивает значения списков, они одинаковые, значит результат True

list1 is list2 → False
объяснение: списки list1 и list2 не ссылаются на один и тот же объект, это разные объекты в памяти

list1 is list3 → True
объяснение: это один и тот же объект в памяти

id(list1) == id(list3) → True
объяснение: идентификаторы объектов list1 и list3 одинаковые, они ссылаются на один и тот же объект в памяти
```
