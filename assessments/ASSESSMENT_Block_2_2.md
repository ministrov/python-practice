## ЧАСТЬ 2: Практическая задача (комплексная)

### Задача: Анализатор заказов интернет-магазина

**Описание:**
Дан список заказов. Каждый заказ — словарь вида:

```python
orders = [
    {"customer": "Anna", "items": ["book", "pen"], "total": 15.5},
    {"customer": "Ivan", "items": ["laptop"], "total": 899.0},
    {"customer": "Anna", "items": ["pen", "notebook"], "total": 8.0},
    {"customer": "Oleg", "items": ["book", "book"], "total": 20.0},
]
```

Напиши программу, которая:

1. Считает **общую сумму заказов по каждому покупателю** (`dict[str, float]`)
2. Собирает **множество всех уникальных товаров**, встречающихся во всех заказах (`set[str]`)
3. С помощью **list comprehension** формирует список покупателей, чей общий заказ превышает `10.0`
4. С помощью **dict comprehension** формирует словарь "покупатель → количество заказов"
5. Выводит результаты в читаемом формате

**Требования:**

- ✅ Использовать `dict` для сумм по покупателю (накопление через `get`/`setdefault`)
- ✅ Использовать `set` для уникальных товаров
- ✅ Хотя бы один `list comprehension`
- ✅ Хотя бы один `dict comprehension`
- ✅ Аннотации типов для переменных, где Pylance strict их требует

**Твой код:**

```python
# YOUR CODE HERE

from typing import cast

# 1. Общая сумма заказов по каждому покупателю
totals_by_customer: dict[str, float] = {}

for order in orders:
    customer: str = cast(str, order["customer"])
    total: float = cast(float, order["total"])
    totals_by_customer[customer] = totals_by_customer.get(customer, 0.0) + total


# 2. Множество всех уникальных товаров
unique_items: set[str] = set()
for order in orders:
    items: list[str] = cast("list[str]", order["items"])
    unique_items.update(items)

# 3. List comprehension: покупатели с суммой заказов > 10.0
big_spenders: list[str] = [
    customer for customer, total in totals_by_customer.items() if total > 10.0
]

# 4. Dict comprehension: покупатель -> количество заказов
orders_count: dict[str, int] = {
    customer: sum(
        1 for o in orders if cast(str, o["customer"]) == customer
    )
    for customer in totals_by_customer
}

# 5. Вывод результатов
print("=== Суммы заказов по покупателям ===")
for customer, total in totals_by_customer.items():
    print(f"{customer}: {total:.2f}")

print("\n=== Уникальные товары ===")
print(", ".join(sorted(unique_items)))

print("\n=== Покупатели с суммой заказов > 10.0 ===")
print(", ".join(big_spenders))

print("\n=== Количество заказов по покупателям ===")
for customer, count in orders_count.items():
    print(f"{customer}: {count}")

```
