## 📖 Как использовать материалы

Каждый блок содержит два файла:

### 1. `XX_topic_demo.py` — Теория
Файл с объяснениями и примерами кода.

```bash
python 2.1_fundamentals/03_variables_demo.py
```

**Что делать:**
- Прочитай объяснения
- Запусти файл и посмотри результаты
- Экспериментируй с кодом

### 2. `XX_topic_task.py` — Практика
Файл с практическими заданиями (с пустыми местами).

```bash
python 2.1_fundamentals/03_variables_task.py
```

**Что делать:**
- Читай условие задания
- Пиши код в место с комментарием `# ТВ КОД ЗДЕСЬ:`
- Запусти файл и проверь результаты
- Если ошибка — исправь код и попробуй снова

## 💻 Примеры из курса

### Переменные и константы
```python
# Переменные
age = 25
name = "Alice"
height = 1.75

# Константы
PI = 3.14159
MAX_ATTEMPTS = 3

# Использование
print(f"{name} is {age} years old")
```

### Работа с типами
```python
# Проверка типа
x = 42
print(type(x))  # <class 'int'>

# Преобразование типа
s = "123"
n = int(s)      # Преобразуем строку в число
```

### Константы в расчётах
```python
PRICE_PER_UNIT = 50.0
TAX_RATE = 0.15
quantity = 4

total = quantity * PRICE_PER_UNIT * (1 + TAX_RATE)
print(f"Total: ${total:.2f}")
```

### Установка Python
Если ещё не установлен Python:
1. Перейди на [python.org](https://www.python.org/)
2. Скачай Python 3.8+ (рекомендуется 3.10+)
3. Установи (не забудь отметить "Add Python to PATH")
