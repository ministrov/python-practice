### Задание 7: Арифметические операции

**Условие:**
```
7.1 Создай переменные a = 15 и b = 3
    Напечатай результаты всех операций:
    - Сложение (a + b)
    - Вычитание (a - b)
    - Умножение (a * b)
    - Деление (a / b)
```

**Решение:**
```python

b = 3

print(a + b)      # 18
print(a - b)      # 12
print(a * b)      # 45
print(a / b)      # 5.0
```

### Задание 18: Константы в расчётах

**Условие:**
```
18.1 Создай константы для расчёта стоимости:
     PRICE_PER_UNIT = 50.0
     TAX_RATE = 0.15
     DISCOUNT = 0.1

18.3 Напечатай итоговую сумму с налогом:
     total = quantity * PRICE_PER_UNIT * (1 - DISCOUNT) * (1 + TAX_RATE)
```

**Решение:**
```python
PRICE_PER_UNIT = 50.0
TAX_RATE = 0.15
DISCOUNT = 0.1
quantity = 4

total = quantity * PRICE_PER_UNIT * (1 - DISCOUNT) * (1 + TAX_RATE)
print(f"Total: ${total:.2f}")  # Total: $216.00
```

## 🎯 Ключевые концепции

### Переменные
```python
x = 10           # Присвоение значения
x = 20           # Изменение значения
x = x + 5        # Использование текущего значения
```

### Типы данных
```python
age = 25                    # int (целое число)
height = 1.75              # float (число с точкой)
name = "Alice"             # str (строка)
is_student = True          # bool (булево значение)
items = [1, 2, 3]          # list (список)
person = {"name": "Bob"}   # dict (словарь)
```

### Константы
```python
PI = 3.14159               # CAPS_LOCK — это константа
MAX_SIZE = 100             # Не меняется во время программы
```

### Форматирование строк
```python
# F-строки (рекомендуется)
print(f"Hello, {name}!")

# Форматирование чисел
print(f"Price: ${price:.2f}")

# Выражения в f-string
print(f"Total: {price * quantity}")
```

## ❓ Часто задаваемые вопросы

**В: Ошибка "NameError: name 'x' is not defined"**
О: Ты использовал переменную до её создания. Сначала создай: `x = 10`

**В: Почему у меня разные результаты?**
О: Проверь, правильно ли значения переменных. Или используешь неправильные формулы.

**В: Как узнать тип переменной?**
О: Используй встроенную функцию: `print(type(x))`

**В: Что такое None?**
О: Это специальное значение, означающее "отсутствие значения"

```python
x = None           # Переменная ничего не содержит
if x is None:
    print("x пуст")
```
