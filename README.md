# Python Backend Developer Journey 🐍

> Практический курс для освоения Python с нуля до backend разработчика

Полный путь обучения от основ синтаксиса до написания масштабируемых backend приложений на Python.

## 🎯 Цель проекта

Этот репозиторий содержит структурированный материал и практические задания для изучения Python с фокусом на backend разработку. Подходит как для самостоятельного обучения, так и для работы с наставником.

## 📚 Структура курса

### 🟢 **01_junior** — Основы (100-150 часов)
Фундаментальные концепции Python, необходимые для любого разработчика.

- **Блок 1**: Основы синтаксиса
- **Блок 2.1**: [Переменные и типы данных](./01_junior/2.1_fundamentals/)
  - 19 практических заданий по переменным, константам, типам
  - Примеры и теория в отдельных файлах
- **Блок 2.2** (планируется): Операторы (условия, циклы)
- **Блок 3** (планируется): Функции и модули
- **Блок 4** (планируется): Работа с данными (списки, словари, множества)

### 🟡 **02_middle** (планируется) — Промежуточный уровень
- Объектно-ориентированное программирование (ООП)
- Работа с файлами и базами данных
- REST API с Flask/FastAPI
- Тестирование кода

### 🔴 **03_senior** (планируется) — Продвинутый уровень
- Архитектура приложений
- Паттерны проектирования
- Асинхронное программирование
- Микросервисы и контейнеризация

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- Git
- Текстовый редактор (VS Code, PyCharm и т.д.)

### Установка

```bash
# Клонируй репозиторий
git clone https://github.com/твой-юзер/python-practice.git
cd python-practice

# Создай виртуальное окружение (опционально, но рекомендуется)
python -m venv venv

# Активируй виртуальное окружение
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
```

### Как использовать материал

Каждый блок содержит:
- **`XX_name_demo.py`** — теория с примерами (читай и запускай)
- **`XX_name_task.py`** — практические задания (решай самостоятельно)

**Рекомендуемый процесс обучения:**
1. Прочитай `_demo.py` и запусти его
2. Изучи примеры кода
3. Решай задания в `_task.py`
4. Проверь свои решения

```bash
# Запуск файла
python 01_junior/2.1_fundamentals/03_variables_demo.py
python 01_junior/2.1_fundamentals/03_variables_task.py
```

## 📖 Примеры заданий

### Переменные и константы (Блок 2.1)

**Задание:** Создай переменные для профиля студента и напечатай информацию

```python
student_name = "Emma"
student_id = 2024001
courses = ["Math", "Physics", "Chemistry"]

print(f"Student {student_name} (ID: {student_id}) studies {len(courses)} courses")
```

**Количество заданий в блоке 2.1:** 19 задач (от базовых до комбинированных)
